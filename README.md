# AI Email Assistant

## Overview
This project is an agentic AI that automatically reads incoming emails and generates professional, context-aware replies based on the email content.

## Features
- Automatically generates polite and professional email responses.
- Maintains context from the incoming email.
- Includes user's name only at the end of the reply.
- Works interactively like a chatbot.

## Technologies Used
- Python
- PyTorch
- Hugging Face Transformers
